# How to Find a Discord Message ID

Message IDs are used for bots that react to or manage specific messages.

**Steps:**

1. Enable **Developer Mode** in Discord settings.
2. Right-click the message and select **Copy Message ID**.

**Tip:** Use message IDs for reaction roles, logging, or moderation.

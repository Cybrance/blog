# How to Host a Replit Bot on Cybrance

1. Export your bot code from Replit.
2. Push it to a GitHub repository.
3. Connect your repo in the Cybrance dashboard.
4. Deploy and manage your bot with Cybrance's hosting solution.

This allows you to move your bot from Replit to a more robust hosting platform.

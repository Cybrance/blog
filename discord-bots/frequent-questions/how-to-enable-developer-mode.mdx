# How to Enable Developer Mode in Discord

Developer Mode lets you copy IDs for roles, channels, users, and more.

**Steps:**

1. Go to **User Settings** > **Advanced**.
2. Toggle **Developer Mode** on.

**Tip:** This is required for most bot setup and troubleshooting tasks.

# How to Find a Discord Guild (Server) ID

Guild IDs are needed for server-specific bot actions and configuration.

**Steps:**

1. Enable **Developer Mode** in Discord settings.
2. Right-click your server icon and select **Copy Server ID**.

**Tip:** Use guild IDs for server-specific features or logging.

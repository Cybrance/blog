# How to Find a Discord User ID

User IDs are useful for bot permissions, direct messaging, and logging.

**Steps:**

1. Enable **Developer Mode** in Discord settings.
2. Right-click the user and select **Copy User ID**.

**Tip:** User IDs are unique and never change, even if the username does.

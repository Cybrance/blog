# Common Discord Bot Errors on Cybrance

## Invalid Token

- Make sure your bot token is correct and up to date.

## Missing Permissions

- Check your bot's role and permissions in the server.

## API Rate Limits

- Avoid sending too many requests in a short time.

## Connection Issues

- Ensure your bot is running and the server is online.

## Environment Variables

- Double-check your secrets and environment variable setup.

# Discord.py Bot Template

This page provides a starter template for building and hosting Discord.py bots on Cybrance.nl. You can use this template to quickly set up your bot and deploy it to GitHub and Cybrance.

## Features
- Python 3.8+
- Discord.py library
- Ready for deployment to Cybrance
- Example GitHub Actions workflow

## Example Bot Code
```python
import discord
from discord.ext import commands
import os

intents = discord.Intents.default()
bot = commands.Bot(command_prefix="!", intents=intents)

@bot.event
async def on_ready():
    print(f"Logged in as {bot.user}")

@bot.command()
async def ping(ctx):
    await ctx.send("Pong!")

bot.run(os.getenv("DISCORD_TOKEN"))
```

## Deploy to GitHub
1. Create a new repository on GitHub and push your bot code.
2. Add your bot token as a GitHub secret named `DISCORD_TOKEN`.

## Example GitHub Actions Workflow
Create `.github/workflows/deploy.yml`:
```yaml
name: Deploy Discord.py Bot
on:
  push:
    branches: [ main ]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.8'
      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install discord.py
      - name: Run bot
        env:
          DISCORD_TOKEN: ${{ secrets.DISCORD_TOKEN }}
        run: |
          python bot.py
```
